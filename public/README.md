# Public Assets

Place your Japanese garden image here.

## Image Setup

1. Add your Japanese garden image to this folder
2. Name it `japanese-garden.jpg`
3. The image will be used as the background with scroll zoom effects

## Recommended Image Specifications

- Format: JPG, PNG, or WebP
- Resolution: 1920x1080 or higher for best quality
- The image will be cropped/zoomed to fit the viewport
